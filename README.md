### FOR TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
> pkg install imagemagick -y
> pkg install libwebp
> git clone https://github.com/lastsymphony/lastv4
> cd lastv4
> npm install
> npm audit fix && npm start
```
###### Run
```bash
> node . [<session name>] (session name is optional)
```
